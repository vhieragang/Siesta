This example showcases HTML5 and CSS3 functionality to visualize a ski cabin rental application. A custom font-family is used (from Google Font API) to show the date in each event. The example also utilizes new features supported in CSS3: transforms (gradient, rotation), rounded corners and box shadow. Best viewed in Firefox, Safari or Chrome.

Note that the js for the example code is not minified so it is readable. See html5.js.
