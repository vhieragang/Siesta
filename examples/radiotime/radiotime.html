This example connects to RadioTime, and loads today's station schedules for the stations in the grid. This is easily done using the RadioTime API using an Ext.data.ScriptTagProxy (JSONP). To listen to any of the channels, click the speaker icon in the left column. This will attempt to open the radio stream in an external player.

Note that the js for the example code is not minified so it is readable. See radiotime.js.
