An example of scheduler with excel-like functionality provided by Sch.plugin.CellPlugin. This example supports following features:

  • Selecting cells
  • Navigating through view using arrows/mouse clicks
  • Cell editing. Just hit enter, double click cell, or simply start typing if cell is already selected

For example, if you want to create event just select cell and type start/end hours separeted by '-' : '10-12'. This will create an event that starts at 10am and ends at 12pm.

Important notice:

this plugin supports only one event per cell. Don't use it with existing event store containing overlapping events.

Note that the js for the example code is not minified so it is readable. See excel.js.
