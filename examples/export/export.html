NOTE: For this example to work you have to run it in a web server context. This example enables you to generate a PDF or PNG file out of a Scheduler panel. It requires having PhantomJS 1.6 and ImageMagick on your server as well as permissions to run them both and create/remove files.

The complete instruction of setting up this example is available in the README.txt file.


Note that the js for the example code is not minified so it is readable. See export.js. You can also take a look into PhantomJs renderer code render.js.
